### Allowed commands

- **gzip** - `.tar.gz` or `.tgz`
- **bzip2** - `.tar.bz2`
- **compress** - `.tar.z`

### Compress files

**gzip** _TAR_FILE_

```sh
$ gzip somefile.tar
```

**bzip2** _TAR_FILE_

```sh
$ bzip2 somefile.tar
```

**compress** _TAR_FILE_

```sh
$ compress somefile.tar
```

### Decompress files

**gunzip** _COMPRESSED_FILE_

```sh
$ gunzip somefile.*
```

**bunzip2** _COMPRESSED_FILE_

```sh
$ bunzip2 somefile.tar.bz2
```

**uncompress** _COMPRESSED_FILE_

```sh
$ uncompress somefile.tar.bz2
```
