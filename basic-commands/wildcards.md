### Wildcards

| Group Symbols | Description                             |
| ------------- | --------------------------------------- |
| \*            | any symbols                             |
| ?             | any one symbol                          |
| [symbols]     | any one of the symbols                  |
| [!symbols]    | any one symbol not of the symbols       |
| [[:class:]]   | any symbol which is belong to the class |

Classes:

- [:alnum:],
- [:alpha:],
- [:digit:],
- [:lower:],
- [:upper:]
