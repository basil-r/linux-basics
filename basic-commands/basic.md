### Finding yourself

```sh
$ pwd
```

### Checking your login

```sh
$ whoami
```

### Navigating

**cd** _PATH_

```sh
$ cd /etc
```

### Listing the content of a directory

**ls** _OPTIONS_

#### Simple list

```sh
$ ls
```

#### Show more info

```sh
$ ls -l
```

#### Show hidden files

```sh
$ ls -la
```

### Getting help

_COMMAND_ **--help** (or **-h**)

```sh
$ git --help
```

```sh
$ git -h
```

### Referencing manual pages

**man** _COMMAND_

```sh
$ man git
```

### List history of commands

```sh
$ history
```
